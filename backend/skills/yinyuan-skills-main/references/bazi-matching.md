| 合 | 化 | 记忆口诀 |

### 地支六冲

两两对冲，冲突明显：

| 冲 | 合婚解读 |
|----|----------|
| 子午冲 | 水火不容，需互相退让 |
| 丑未冲 | 同类竞争，容易较劲 |
| 寅申冲 | 金木交战，意见分歧大 |

**注意**：六冲不代表一定不合，需结合其他因素综合判断。冲有时反而激荡出火花。

## 3. 五行生克

### 五行相生

木 → 火 → 土 → 金 → 水 → 木

（生：滋养、促进、帮助）

### 五行相克

木 → 土 → 水 → 火 → 金 → 木

（克：制约、压制、冲突）

### 日主五行关系

以双方日干（日柱天干）判断两人关系：

| 关系 | 含义 | 感情解读 |
|------|------|----------|
| 比肩 | 同类 | 像朋友一样平等相处 |
| 劫财 | 同类但竞争 | 既亲密又较劲 |
| 食神 | 我生 | 一方付出多，照顾另一方 |
| 伤官 | 我生但带克 | 热烈但容易摩擦 |
| 偏财 | 我克 | 一方主导，一方被吸引 |
| 正财 | 我克但柔 | 温和的吸引，稳定配对 |
| 七杀 | 克我 | 一方强势，需要磨合 |
| 正官 | 克我但柔 | 被保护的感觉，传统良配 |
| 偏印 | 生我 | 被照顾但有压力 |
| 正印 | 生我且柔 | 温暖的包容，安全感的来源 |

## 5. 综合评分规则

### 评分维度与权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 天干关系 | 15% | 年干、日干的合冲关系 |
| 地支六合/三合 | 20% | 年支的合局关系 |
| 地支六冲/六害 | 15% | 年支的冲害关系（扣分项） |
| 日柱配合 | 25% | 日主五行关系 + 日支互动 |
| 五行互补 | 15% | 双方八字五行的互补程度 |
| 纳音配对 | 10% | 年柱纳音的生克关系 |

### 评分参考

| 分数区间 | 等级 | 解读 |
|----------|------|------|
| 90-100 | 天作之合 | 姻缘极深，好好珍惜 |
| 75-89 | 佳偶天成 | 底子很好，用心经营可长久 |
| 60-74 | 尚可配对 | 有缘但需磨合，多沟通 |
| 45-59 | 需要用心 | 挑战较大但非不可，重在经营 |
| 30-44 | 有缘无分 | 缘分较浅，随缘为好 |
| 0-29 | 不建议强求 | 天然冲突较多，在一起需极大努力 |

**注意**：分数只是参考工具，不是判决书。任何分数区间都加上"经营"和"成长"的建议。
