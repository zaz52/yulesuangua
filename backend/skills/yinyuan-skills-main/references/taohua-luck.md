## 2. 桃花类型判断

### 正桃花

**特征**：带来真诚的感情，可能发展为稳定的伴侣关系
**表现**：
- 遇到的人真诚可靠
- 相处舒服自然
- 双方都愿意投入
- 家人朋友认可

### 偏桃花

**特征**：有吸引力但不够稳定，可能是短暂的恋情
**表现**：
- 一见钟情型，来势汹汹
- 初期甜蜜后期暴露问题
- 有感觉但不踏实
- 外界不太看好

### 桃花劫

**特征**：因感情带来损失（财务、精力、名誉等）
**表现**：
- 感情中花销大、精力耗损多
- 为感情影响工作和学业
- 周围人劝你但听不进去
- 分合反复，身心俱疲

### 桃花煞

**特征**：感情中带伤害（欺骗、背叛、纠纷等）
**表现**：
- 对方有隐瞒或欺骗行为
- 卷入三角关系
- 感情引发人际纠纷
- 分手后心伤较重

### 判断规则

| 条件 | 倾向 |
|------|------|
| 桃花星+吉星（禄存/魁钺/辅弼） | 正桃花 ✅ |
| 桃花星+煞星（羊/陀/火/铃） | 桃花劫 ⚠️ |
| 桃花星+化忌 | 桃花煞 ❌ |
| 桃花星+空劫 | 虚桃花（镜花水月） |
| 桃花星+文曲/文昌 | 浪漫型桃花 |
| 桃花星+天喜/红鸾 | 正缘桃花 |

### 流年桃花星查法

流年桃花 = 流年地支查三合局对应桃花位：

- 流年寅午戌 → 卯（正东）
- 流年申子辰 → 酉（正西）
- 流年巳酉丑 → 午（正南）
- 流年亥卯未 → 子（正北）

2026年为午年，属于寅午戌火局，流年桃花在卯（正东）。

## 5. 桃花方位与开运

### 桃花方位速查

| 需求 | 方法 |
|------|------|
| 提升桃花运 | 在桃花位放置鲜花（真花）或粉色物件 |
| 增加社交机会 | 多去桃花方位的活动/聚会 |
| 约会选位 | 选在双方的共同桃花方位最佳 |
| 化解烂桃花 | 在桃花位放铜器或金属物件（金克木，制桃花） |

### 桃花颜色

| 五行 | 颜色 | 适合谁 |
|------|------|--------|
| 木 | 绿色 | 缺木的人 |
| 火 | 红色/粉色 | 缺火的人 |
| 土 | 黄色/棕色 | 缺土的人 |
| 金 | 白色/金色 | 缺金的人 |
| 水 | 蓝色/黑色 | 缺水的人 |

穿自己五行所缺的颜色，有助于提升整体运势包括桃花运。

## 8. 解读风格

- 正桃花：语气积极鼓励
- 偏桃花：温和提醒，不打击
- 桃花劫/煞：认真提醒，但不过度恐吓
- 所有类型都以"你可以做什么"结尾，而不是"你完蛋了"
