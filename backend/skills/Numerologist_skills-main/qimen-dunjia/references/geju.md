# 常见格局与阅读方式

这个文件不是让你把所有古诀都原样倒给用户，而是帮助你把格局说成人话。

## 先看顺序

正式解盘时，格局判断按这个顺序看：

1. 先看主用神本身落宫好不好
2. 再看值符、值使是否帮扶
3. 再看门、星、神的组合
4. 最后才看特殊格局名目

如果主用神已经明显受制，就不要被个别“吉名”带偏。

## 用户容易听懂的吉象

### 三奇配吉门

当乙、丙、丁这类三奇配上开门、休门、生门时，通常可解释为：

- 事情有转机
- 沟通、行动、资源调动更顺
- 更适合推进、见人、签约、求助

### 值符得位

值符得位时，通常可解释为：

- 主导权更稳
- 容易得到关键人物支持
- 事情更容易走到明面和正面路径

### 生门、开门得力

- 生门偏资源、财气、恢复力
- 开门偏事业、公开推进、权威支持

## 用户需要警惕的凶象

### 门受制

如果关键门被所在宫位所克，优先翻译成：

- 条件不顺
- 推进容易受卡
- 现在硬冲，成本会高

### 旬空

旬空不直接等于“坏”，更适合解释为：

- 现在力量虚
- 容易落空、拖延、迟到
- 需要等时机落地

### 白虎、玄武干扰

- 白虎重时，提醒冲突、伤损、硬碰硬代价
- 玄武重时，提醒信息失真、反复、隐情、小人、拖账

## 输出原则

- 格局名可以提，但必须翻译成用户能执行的判断。
- 不要用格局名替代分析。
- 同时有吉有凶时，要说明“哪一块能做，哪一块先别做”。
