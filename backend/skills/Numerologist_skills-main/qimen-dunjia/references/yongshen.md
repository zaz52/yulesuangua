# 用神取用顺序

正式解盘不要一上来堆很多用神。先抓主问题，再确定主用神，最后再加辅助参考。

## 总顺序

1. 先确认“谁是主体”
2. 再确认“要判断什么结果”
3. 先定主用神
4. 再加 1 到 2 个辅助参考

## 基础定位

### 求测者本人

- 优先看日干落宫

### 所问之事 / 对方 / 当前动作

- 优先看时干落宫

## 常见事项

### 事业 / 工作 / 项目推进

- 主用神：开门
- 辅助：值符、日干

先看：

- 开门得不得力
- 日干和开门关系如何
- 值符是否给到主导权和支持

### 求财 / 交易 / 投资

- 主用神：生门
- 辅助：戊、日干

先看：

- 生门有没有被克
- 戊的位置是否稳
- 日干能不能接住这笔财

### 感情

先问清楚求测者的性别和对象性别，再确定用神：

- 男问女（或问异性）：乙为主要参考
- 女问男（或问异性）：庚为主要参考
- 同性关系：优先取六合为主用神，辅助看日干与时干的关系
- 辅助：六合、日干

如果用户没有明说对象性别，先补问，不要擅自假设。

### 考试 / 学习 / 证书

- 主用神：景门、天辅
- 辅助：丁

### 疾病 / 健康

- 主用神：天芮、死门
- 辅助：日干

输出时必须加现实提醒：

- 奇门只能看趋势，不代替就医和检查

### 出行 / 方位 / 见人

- 主用神：目标方位所在宫
- 辅助：开门、生门、值符

### 诉讼 / 合同 / 争议

- 主用神：开门、惊门
- 辅助：天心、值符

### 寻人 / 寻物

- 主用神：时干 + 类象
- 辅助：玄武、门的状态

## 解读方法

写用神分析时按这个模板：

1. 这次主用神我取什么
2. 我为什么这样取
3. 它现在落在哪一宫
4. 这宫里的门、星、神是帮它还是压它
5. 这对用户最关心的问题意味着什么
