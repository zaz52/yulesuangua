## 示例 1：正式排盘（跳槽）

### 访谈确认信息

- 事项：跳槽/换工作
- 起局时间：2026-03-24 10:30（公历）
- 地点：上海（China/Asia/Shanghai）
- 目标：能不能动，什么时候动更稳
- 输出偏好：直接结论

### 脚本关键输出（摘要）

执行脚本后从输出 JSON 提取以下关键字段：

```

  year:  丙午
  month: 辛卯
  day:   丁酉
  time:  乙巳
  time_xun:     甲辰
  time_xunkong: 寅卯

  dun_type:          阳遁

  - 本规则集中宫相关判断一律寄坤处理。
```

### 读盘过程（内部，不展示给用户）

1. **主用神**：事业/跳槽 → 取开门为主用神，辅助看值符和时干（乙）。
2. **开门落宫**：7宫兑宫（西），配天心/太阴。天心主聪明谋略，太阴主隐秘暗动 → 开门得神配合，但属暗动、低调路线，不适合大张旗鼓。
3. **用户代表（时干乙）**：落 4宫巽宫（东南），配景门/九天 → 用户目前视野开阔、动机强，但景门偏中等，执行力不是最强。
4. **值符值使**：同落 9宫离宫（南），死门 + 天芮 → 死门不利主动推进，天芮主迟滞/消耗，当前大格局偏保守。
5. **旬空**：寅卯对应 3宫（东）和 8宫（东北）→ 这两个方向力量虚，今日往东或东北向谈事效果打折。
6. **核心矛盾**：开门本身条件不差，但值使死门压着，时机略显保守；事不急则可动，急则容易在细节上受卡。

## 示例 2：理论学习模式

### 场景

用户说：值符和值使是什么意思？

### 处理方式

不进入正式排盘。先确认学习需求，再解释：

---

**值符**（zhífú）：当局的"主导星神"，代表当前时间段最有权威、最有主导力的力量方向。它的落宫决定了整个盘里"谁说了算"，是判断主动权和支持来源的关键参考。

**值使**（zhíshǐ）：当局的"执行门"，代表当前最活跃的行动通道。它的门名（休/生/伤/杜/景/死/惊/开）直接影响当前适合做什么事。

**两者关系：**

- 值符和值使通常一起看——如果两者落在同一宫，说明"主导力量"和"行动通道"高度集中，效率高但也意味着其他宫位相对弱。
- 如示例 1，值符（天芮）和值使（死门）同在离宫（南），死门主消耗收尾，说明当前时间段不适合激进开拓。
