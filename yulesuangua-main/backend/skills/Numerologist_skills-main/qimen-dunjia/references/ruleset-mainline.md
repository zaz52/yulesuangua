# `mainline-cn-v1` 默认规则

这个 skill 的正式排盘只使用这一套默认规则，不在正式计算里混用别的说法。

## 适用范围

- 面向中国大陆常见时家转盘奇门使用习惯
- 默认时区：`Asia/Shanghai`
- 海外用户必须先补城市或时区

## 固定规则

### 1. 时间规则

- 用户说“现在”，就按当前绝对时间起局。
- 没给时区且语境是中国大陆，默认北京时间。
- 海外用户没给时区时，先追问，不要直接排盘。

### 2. 节令规则

- 以节令切换阴遁 / 阳遁。
- 冬至到芒种：阳遁
- 夏至到大雪：阴遁
- 节气边界前后 24 小时内，输出中要明确写出“边界提醒”。

### 3. 三元规则

- 按日干支在六十甲子序列中的位置分组。
- 每 5 日为一元。
- 顺序固定循环为：上元 -> 中元 -> 下元。

这是当前版本采用的工程化固定规则，用来确保同一输入得到同一结果。

### 4. 定局规则

- 阴遁 / 阳遁与三元确定后，从固定定局表直接取局数。
- 不在正式排盘时临时切换拆补法或其他变体。

### 5. 中宫规则

- 中宫相关判断一律寄坤处理。
- 如果旬首所遁之仪或时干落在中宫，正式输出里要提示“本盘按寄坤处理”。

### 6. 展示规则

- 不展示完整推理链。
- 只展示关键计算结果和必要依据。
- 用户主要看到的是：阴阳遁、局数、旬首、旬空、值符、值使、关键用神。

## 不支持的情况

以下情况不进入正式排盘：

- 用户明确要求切换到别的规则，但不接受当前默认规则
- 缺少时间或地点信息
- 想让模型跳过脚本直接手算
