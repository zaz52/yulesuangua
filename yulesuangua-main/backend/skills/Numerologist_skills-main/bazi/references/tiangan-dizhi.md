## 大运排列规则

- 阳年（甲丙戊庚壬）男 / 阴年（乙丁己辛癸）女 → 顺排
- 阴年男 / 阳年女 → 逆排
- 以月柱干支为基准，按六十甲子顺序往后/往前推
- 起运年龄 = 出生日至下一个（顺排）或上一个（逆排）节气的天数 ÷ 3
