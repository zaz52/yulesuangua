## 早子时与夜子时

子时横跨两天（23:00-01:00），在命理学中有"早子时"和"夜子时"之分：

- **早子时**（00:00 - 01:00）：归属当日，使用当日日柱
- **夜子时**（23:00 - 24:00）：归属次日，使用次日日柱

注：子时归属是命理学长期争议的话题。本 skill 采用"早晚子时"区分法，即：
- 23:00 前按当日算
- 23:00 后按次日日柱计算时柱

| 天快黑 / 傍晚 | 酉时 |
| 吃晚饭 / 晚上 | 戌时 |
| 夜里 / 睡前 | 亥时 |
| 半夜 / 深夜 | 子时 |

## 日上起时法（五鼠遁元）

时柱天干由日干决定，口诀：

> 甲己还加甲，乙庚丙作初，丙辛从戊起，丁壬庚子居，戊癸何方发，壬子是真途。

即日干为甲或己时，子时天干为甲（甲子时）；日干为乙或庚时，子时天干为丙（丙子时）；以此类推。
