## 天干五合

甲己合化土
乙庚合化金
丙辛合化水
丁壬合化木
戊癸合化火

合化条件：需得时得令，天干相邻且地支有合化之根，方论化。仅合不化则为"合绊"，力量牵制。

## 地支关系

### 六冲
子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲

### 三合局
- 申子辰合水局
- 亥卯未合木局
- 寅午戌合火局
- 巳酉丑合金局

三合局需三字全见方成局，缺一则为半合（如申子半合水、子辰半合水），力量减半。

### 三会局（方局）
- 寅卯辰会东方木局
- 巳午未会南方火局
- 申酉戌会西方金局
- 亥子丑会北方水局

三会局力量大于三合局，需三字全见。

### 六合
子丑合土、寅亥合木、卯戌合火、辰酉合金、巳申合水、午未合火（或合土，有争议）

### 相刑
- 寅巳申三刑（无恩之刑）
- 丑戌未三刑（持势之刑）
- 子卯相刑（无礼之刑）
- 辰辰自刑、午午自刑、酉酉自刑、亥亥自刑

### 相害（相穿）
子未害、丑午害、寅巳害、卯辰害、申亥害、酉戌害

## 十神定义及生克关系

### 十神推导规则
以日干（日主）为中心，根据其他天干与日干的五行生克关系及阴阳异同来确定十神：

| 关系 | 阴阳相同 | 阴阳不同 |
|------|---------|---------|
| 生我（我之母） | 偏印（枭神） | 正印 |
| 我生（我之子） | 食神 | 伤官 |
| 克我（克我者） | 偏官（七杀） | 正官 |
| 我克（我制者） | 偏财 | 正财 |
| 同我（同类） | 比肩 | 劫财 |

### 十神详解

| 十神 | 含义 | 代表人事 |
|------|------|---------|
| 比肩 | 同我者，阴阳相同 | 兄弟、朋友、同辈、竞争者 |
| 劫财 | 同我者，阴阳不同 | 姐妹、竞争对手、损财者 |
| 食神 | 我生者，阴阳相同 | 子女（女命）、才华、口福 |
| 伤官 | 我生者，阴阳不同 | 子女（女命）、才艺、叛逆 |
| 偏财 | 我克者，阴阳相同 | 偏财、父（男命）、情人 |
| 正财 | 我克者，阴阳不同 | 正财、妻（男命）、稳定收入 |
| 偏官（七杀） | 克我者，阴阳相同 | 偏权、压力、小人之类 |
| 正官 | 克我者，阴阳不同 | 正权、上级、夫（女命） |
| 偏印（枭神） | 生我者，阴阳相同 | 偏学术、继母、孤独 |
| 正印 | 生我者，阴阳不同 | 母亲、学业、贵人 |

### 五行生克
- 相生：木→火→土→金→水→木
- 相克：木→土→水→火→金→木
