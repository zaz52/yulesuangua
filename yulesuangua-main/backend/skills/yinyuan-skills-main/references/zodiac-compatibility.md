## 说明

生肖配对是八字合婚的简化版本，仅凭两个人的出生年份（地支/生肖）即可做初步判断。适合信息不足时的快速配对，或作为趣味参考。

## 2. 六合（最佳配对）

六合是地支两两暗合，配对指数最高：

| 配对 | 解读 |
|------|------|
| 鼠 + 牛 | 子丑合化土。鼠的机灵配上牛的踏实，互补型佳偶。鼠提供灵感，牛负责落地。 |
| 虎 + 猪 | 寅亥合化木。虎的霸气遇上猪的温柔，刚柔并济。猪能包容虎的脾气。 |
| 兔 + 狗 | 卯戌合化火。兔的细腻遇到狗的忠诚，相知相守。两人都很重感情。 |
| 龙 + 鸡 | 辰酉合化金。龙的远见配上鸡的精细，互相欣赏。是彼此的头号粉丝。 |
| 蛇 + 猴 | 巳申合化水。蛇的深沉遇上猴的灵动，棋逢对手。精神层面很契合。 |
| 马 + 羊 | 午未合化土。马的热情遇上羊的温顺，温暖相伴。天生一对欢喜冤家。 |

**配对指数：⭐⭐⭐⭐⭐（95分）**

## 3. 三合（极佳配对）

三合是三个生肖组成一个合局，两两之间都有极好的默契：

### 申子辰 · 水局（猴鼠龙）

| 配对 | 解读 |
|------|------|
| 猴 + 鼠 | 水上双子，默契满分。都是聪明人，沟通零障碍。 |
| 猴 + 龙 | 灵猴配神龙，互相激发创造力。做情侣做搭档都强。 |
| 鼠 + 龙 | 鼠的精细配上龙的格局，小处大处都能打配合。 |

### 寅午戌 · 火局（虎马狗）

| 配对 | 解读 |
|------|------|
| 虎 + 马 | 热情碰撞，生活不会无聊。行动力爆表的一对。 |
| 虎 + 狗 | 虎的霸气配上狗的忠义，义气相投的伴侣。 |
| 马 + 狗 | 马的自由加上狗的守候，一个跑一个等，最终殊途同归。 |

### 巳酉丑 · 金局（蛇鸡牛）

| 配对 | 解读 |
|------|------|
| 蛇 + 鸡 | 精英组合，审美一致。两人都有追求完美的倾向。 |
| 蛇 + 牛 | 蛇的智慧配上牛的勤恳，闷声发大财型情侣。 |
| 鸡 + 牛 | 鸡的条理配上牛的稳定，日子过得井井有条。 |

### 亥卯未 · 木局（猪兔羊）

| 配对 | 解读 |
|------|------|
| 猪 + 兔 | 温柔乡里两颗心，岁月静好型配对。 |
| 猪 + 羊 | 两个温和的灵魂，生活中充满善意和体贴。 |
| 兔 + 羊 | 兔的优雅遇上羊的浪漫，文艺清新型恋人。 |

**配对指数：⭐⭐⭐⭐（85分）**

## 4. 六冲（冲突配对）

六冲是正对面的生肖，天然存在对立面：

| 配对 | 冲突点 | 化解建议 |
|------|--------|----------|
| 鼠 + 马 | 子午冲。水火对冲。鼠重细节，马重自由；鼠慢热，马奔放。 | 给彼此空间，尊重差异。中间人/生肖：虎（三合马，与鼠不冲） |
| 牛 + 羊 | 丑未冲。同类相斥。都固执，冷战高手。 | 学会主动低头，轮流让步。中间人/生肖：蛇或鸡（三合牛） |
| 虎 + 猴 | 寅申冲。金木交战。虎直接，猴灵活；都爱主导。 | 明确分工，不要抢同一个方向盘。中间人/生肖：鼠（三合猴） |
| 兔 + 鸡 | 卯酉冲。金克木。兔感性，鸡理性；节奏不同频。 | 多做对方喜欢的事，用行动弥补频率差。中间人/生肖：牛或蛇（三合鸡） |
| 龙 + 狗 | 辰戌冲。同性相斥。都有主见，容易较劲。 | 分清"大事"和"小事"，小事让步大事商量。 |
| 蛇 + 猪 | 巳亥冲。水火对冲。蛇深沉，猪直爽；猜疑 vs 透明。 | 保持坦诚沟通，减少内心戏。中间人/生肖：兔或羊（三合猪） |

**配对指数：⭐⭐（50分，但并非不可调和）**

**重要提醒**：六冲不代表一定不合！很多恩爱夫妻生肖对冲，反而在冲撞中找到平衡。关键在于是否愿意磨合。

## 5. 六害（暗损配对）

六害关系不如六冲那么激烈，但容易在日常中积累不满：

| 配对 | 问题 | 建议 |
|------|------|------|
| 鼠 + 羊 | 子未害。关心方式不被对方接受 | 多问对方需要什么，不要"自以为好"地付出 |
| 牛 + 马 | 丑午害。节奏不一致，牛慢马快 | 找到双方都舒适的节奏，不强求同步 |
| 虎 + 蛇 | 寅巳害。好意容易被误解 | 表达时注意方式，减少误解空间 |
| 兔 + 龙 | 卯辰害。小事积累矛盾 | 及时沟通小问题，不要攒着 |
| 猴 + 猪 | 申亥害。信任容易出裂痕 | 建立透明的沟通机制，不要让对方猜 |
| 鸡 + 狗 | 酉戌害。表面和谐暗有分歧 | 定期深入交流，不要只聊表面 |

**配对指数：⭐⭐⭐（60分）**

## 8. 解读风格指南

### 配对好的

- 强调默契点和互补优势
- 提醒不要因为"很配"就忽视经营
- 给出保持甜蜜的建议

### 配对一般的

- 强调关系中性的底色，好坏看经营
- 指出可能的摩擦点
- 给出具体可行的建议

### 配对有冲的

- 先安抚：冲不代表不合
- 分析冲的具体表现
- 给出化解方向
- 语气温和，不要制造焦虑
