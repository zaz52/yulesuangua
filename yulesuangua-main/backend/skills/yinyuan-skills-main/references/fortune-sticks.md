# 姻缘签诗系统

## 说明

本文件提供一套完整的姻缘签诗系统。签诗融合传统签诗的意境和现代感情解读，每支签包含签诗、签解、姻缘解读三个层次。

---

## 签号生成规则

签号由以下因素综合生成，保证同一天同一人对同一问题得到同一支签：

```
签号 = (名字字符编码和 + 问题描述字符编码和 + 当前时辰序号 + 当前日期天数) % 100 + 1
```

范围：1-100

---

## 签运分级

| 等级 | 签号范围 | 含义 |
|------|----------|------|
| 上上签 | 1-10 | 天赐良缘，好好把握 |
| 上签 | 11-30 | 姻缘将至，保持期待 |
| 中上签 | 31-45 | 有缘有分，稍加努力 |
| 中签 | 46-55 | 缘分平平，顺其自然 |
| 中下签 | 56-70 | 有些坎坷，但非绝路 |
| 下签 | 71-85 | 时运未到，先修自身 |
| 下下签 | 86-100 | 暂不宜动，静待转机 |

### 上上签（1-10）

**第一签 · 上上**
```
    月老亲牵红线来，
    前世姻缘今世开。
    莫愁前路无知己，
    有情之人自会来。
```
签解：此签大吉。月老亲临，前世有约今生来赴。无论目前状态如何，正缘已在路上。
姻缘解读：你的姻缘运势极佳。如果是单身，近期有极大可能遇到心动之人；如果已有目标，大胆表白成功率很高。

**第二签 · 上上**
```
    百年好合在今朝，
    金玉良缘不用求。
    春风得意马蹄疾，
    一日看尽长安花。
```
签解：此签大吉。金玉良缘天注定，无需刻意追寻。春风正暖，万物可期。
姻缘解读：不用焦虑，不用刻意。你的正缘会在最自然的状态下出现，越是放松越容易遇到。

**第三签 · 上上**
```
    双燕归来春正暖，
    桃花映面柳含烟。
    此情只应天上有，
    人间难得几回闻。
```
签解：双燕齐飞，桃花盛开。姻缘之美，如天上人间。
姻缘解读：一段非常美好的感情正在向你走来。如果已经有人在身边，这段感情会更加甜蜜。

**第四签 · 上上**
```
    缘定三生石上书，
    今朝得见旧时图。
    莫道相逢皆偶然，
    天机早在命中铺。
```
签解：三生石上有名，此缘非偶然。命中注定的重逢。
姻缘解读：你遇到或即将遇到的那个人，和你有很深的缘分。可能有似曾相识的感觉。

**第五签 · 上上**
```
    东风送暖百花开，
    喜鹊枝头报喜来。
    莫问姻缘何处觅，
    回头一笑在身旁。
```
签解：东风送暖，喜报频传。回眸之间，缘分就在身边。
姻缘解读：注意身边的人，也许你一直在寻找的那个人就在你周围。

**第六签 · 上上**
```
    红线牵来天作合，
    两心相悦不需说。
    但愿人长久万里，
    共看明月照山河。
```
签解：天作之合，心意相通。千里共婵娟。
姻缘解读：你和对方之间有一种不需言语的默契，这是极好的姻缘征兆。

**第七签 · 上上**
```
    金风玉露一相逢，
    便胜却人间无数。
    两情若是久长时，
    又岂在朝朝暮暮。
```
签解：取自秦观名句。金风玉露之逢，胜过人间无数。
姻缘解读：即使目前是异地或有距离，真挚的感情不受距离限制。

**第八签 · 上上**
```
    明月照人圆又满，
    清风送来桂花香。
    此生得遇知心人，
    不枉来人间一趟。
```
签解：月圆花香，得遇知心。人生圆满之象。
姻缘解读：你即将遇到一个非常懂你的人，能读懂你的沉默，理解你的选择。

**第九签 · 上上**
```
    春江水暖鸭先知，
    你心已动缘已至。
    不必犹豫不必等，
    勇敢迈出第一步。
```
签解：春水先知，你的直觉是对的。
姻缘解读：如果你心里已经有了感觉，相信自己的直觉。对方可能也有同样的心思。

**第十签 · 上上**
```
    天生一对地一双，
    命里姻缘不可挡。
    纵使千山万水隔，
    终究会到彼此旁。
```
签解：命中姻缘，纵有千难万阻终能团圆。
姻缘解读：如果目前有困难，不要放弃。时间和真心会解决一切障碍。

## 解读风格指南

### 上上/上签

- 语气积极温暖
- 鼓励行动
- 提醒珍惜和经营

### 中签

- 语气平和中肯
- 指出当前状态
- 给出务实建议

### 中下/下签

- 先共情安抚
- 把凶象转化为提醒和建议
- 始终留有希望和转机
- 下下签的结尾一定要有积极的转机暗示
- 特别注意：下下签绝不等于"完蛋了"

### 所有签的结尾统一加

> 签诗仅供娱乐参考，心诚则灵。同一问题今日不宜再签，给缘分一点时间。
